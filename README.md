# Build Lumina: A Daily Quote API

### Project Description

You are going to build Lumina, a daily quote and affirmation API using Django and Django REST Framework. You will start by setting up a simple database to store inspirational quotes, author names, and categories. Then, you will bring the API to life by writing custom views that handle filtering and searching manually. You will learn how to use DRF's base `APIView`, extract data from `request.query_params`, and use Django's built-in ORM to query the database. This hands-on approach teaches you exactly how HTTP requests map to database actions, enabling frontend apps to effortlessly request something specific like `/api/quotes/?category=focus`.

## Tech Stack

- Django REST Framework
